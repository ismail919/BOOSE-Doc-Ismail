AppCanvas(int, int)

Initializes a new instance of the AppCanvas class with the specified dimensions.

public AppCanvas(int xsize, int ysize)

Parameters

xsize int

Canvas width in pixels. Should be a positive integer.

ysize int

Canvas height in pixels. Should be a positive integer.

Remarks

Creates an off-screen Bitmap of the requested size and a Graphics surface for drawing. The initial pen is created with Red and the pen position is initialized to (120, 120).

Remarks

The getter returns the current Color boxed as an object. The setter expects a Color instance (boxed as object); invalid types are ignored and an error message is written to the console.

Parameters

Radius of the circle in pixels. Values >= 0 are expected.

filled bool

If true, the circle is filled with the current pen color. If false, an outline is drawn. Note: current implementation always attempts to fill and falls back to outline on error.
